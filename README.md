# WISE.UWWTD Public

A repository of publicly available data related to the Urban Waste Water Treatment Directive (UWWTD).  
This repository is intended to supplement and make it easier to work with the datasets already provided by official sources.

## Data Sources

The primary data originates from the European Environment Agency (EEA):

- UWWTD Data Registers  
  https://discomap.eea.europa.eu/uwwtd/registers/

- EEA Metadata Catalogue Entry  
  https://sdi.eea.europa.eu/catalogue/srv/eng/catalog.search#/metadata/6244937d-1c2c-47f5-bdf1-33ca01ff1715

## Purpose

This repository aims to:
- Provide easier access to selected datasets
- Enable reuse and analysis of UWWTD-related data
- Offer structured or transformed versions of the original data where useful
